# Features

This repository supports the personalDNSfilter Android app. The key features include:

- **DNS Filtering**: Block ads and unwanted hosts based on Host Name filtering.
- **Android App Support**: Only the Android version is currently supported, allowing users to configure the DNS proxy directly on their Android devices.
- **Ad Blocking**: Block malicious or tracking hosts efficiently via the DNS protocol.
- **Custom Filters**: Ability to add custom host filtering configurations via the Android App UI.
